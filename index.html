Features:

  • Create an infinite number of profiles, each with an infinite number of services, each of which with an infinite number of accounts.
  • Cross-platform, coded in C# and fully functional in both .NET and Mono. PinCushion will run and behave in the exact same way across the following operating systems: Windows, Linux, MacOS.
  • Data is encrypted both on disk and in memory with AES 256-bit in conjunction with SHA-512 hashes.
  • Very intuitive, easy to use interface.

Getting PinCushion

Getting PinCushion is very simple, simply go to this page and download the version appropriate to your OS and architecture. There are 32-bit and 64-bit downloads for Windows, Linux and MacOS. Simply download it and place it in a folder of your chosing, there is no installation procedure involved as it is a single-executable product.

Compiling PinCushion

For those interested in a more hands-on approach, compiling PinCushion should be relatively straightforward.

  1. From this page either clone the project using Git or just download the .ZIP.
  2. Open the solution (PinCushion.sln) in your favourite IDE (it has been compiled successfully in both Visual Studio Express as well as MonoDevelop).
  3. From the configurations menu (where you are able to select which configuration to use for the build) pick whichever platform you are building it for.
    • .NET or Mono (both available in AnyCPU and x64)
  4. Build it and you should be able to find the executable in the PinCushion/bin subfolder of wherever you cloned/extracted PinCushion to.
  5. Copy that executable to wherever you want and run it.
  6. Create a shortcut for it in whatever way you feel comfortable with and start using it, as there is no installation procedure involved.
